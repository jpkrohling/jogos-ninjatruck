# NinjaTruck

Um jogo criado por Peter, 6 anos.

Dirija um monster truck e não deixe os ninjas subirem! Desvie, use o turbo e sacuda pra se livrar deles.

## Como jogar

- **Setas / WASD** — mover o truck
- **Espaço** — turbo (a partir do nível 2)
- **E / Shift** — sacudir inimigos (a partir do nível 3)
- **P** — pausar
- No celular: joystick virtual + botões touch

## Temas de inimigos

Escolha entre Ninjas, Tartarugas Ninjas, Ninjago, Gummibärchen ou Pokémons!
